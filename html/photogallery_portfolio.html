ПОРТФОЛІО АКТОРА
перепустка у світ мистецтва

Акторське портфоліо необхідне і дорослим, і дітям - всім, хто мріє спробувати себе у ролі актора або учасника телешоу. Наявність портфоліо з повною палітрою портретів вашого емоційного діапазону дозволить вам вигідно вирізнитися вже на попередньому етапі кастингу.

Одяг - в жодному разі не over-size, щоб адекватно оцінити статуру. Перевагу віддавайте нейтральним, спокійним кольорам та не надто активним принтам. Одяг має бути зручним!
Для хлопчиків: джинси/штани (не треники!) + сорочка/товстовка/футболка з довгим рукавом.
Для дівчаток: джинси + світшот/футболка з довгим рукавом, сукня без оздоблення (футляр, пряме, шкільний сарафан), базова спідниця, у т.ч. шкільна + спокійний верх.

Волосся - як Ви зазвичай носите, головне - чисте та акуратно висушене. Не робіть складних зачісок, кучерів, супер-кіс.

Макіяж - категорично ні! Допустимо легкий тональний засіб, туш для вій. Обов'язково для ВСІХ - зволоження губ гігієнічною помадою, акуратний манікюр.

Обов'язково: змінне взуття з чистою підошвою (бажано із закритим носком), гребінець.

P.S. Найголовніше - гарний настрій. Позитивне налаштування під час фотозйомки - 80% успіху!!!
